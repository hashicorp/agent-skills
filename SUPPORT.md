# Support

Repository support covers Skill content, installation guidance, marketplace
metadata, plugin packaging, and reproducible Skill behavior. Open a public issue
only for non-sensitive repository problems and use the matching Markdown
template.

Repository maintainers do not provide:

- HashiCorp product support or incident response;
- model-provider service, billing, or model-quality support;
- Claude Code, Codex, or another coding harness's product support;
- troubleshooting for a user's local network, credentials, operating system,
  cloud account, or production environment.

Use the applicable product or provider's established support path for those
concerns. Do not share credentials or sensitive infrastructure details. Follow
[SECURITY.md](SECURITY.md) for any sensitive report.

Support has no time-based response SLA. `@hashicorp/team-agent-skills-ecosystem`
owns final repository decisions; product-aligned owners may participate in
product-correctness review when confirmed in `CODEOWNERS`.
